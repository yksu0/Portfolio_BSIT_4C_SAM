# Personal Portfolio Website

## Project Overview

This project is a **personal portfolio website** developed to showcase my skills, projects, and background as a developer. The website will present my work in a **clean, visually appealing, and interactive layout**, allowing visitors to easily learn about me and explore my projects.

The portfolio will also demonstrate the use of **modern web development tools and technologies**, including a modern frontend framework, animations, and a cloud backend.

---

# Project Goals

The goal of this project is to create a **professional and responsive portfolio website** that:

* Introduces who I am as a developer
* Highlights my technical skills
* Showcases my projects and sample works
* Provides a way for visitors to contact me
* Demonstrates modern UI design and animations

The site will be deployed online and the full source code will be available through a GitHub repository.

# Project Workflow

The development process will follow these steps:

1. Set up the Next.js project environment
2. Configure Tailwind CSS for styling
3. Design the main page layout
4. Create reusable components (Navbar, Skills, Projects, Contact)
5. Implement animations and transitions
6. Connect the contact form to Supabase
7. Test responsiveness and usability
8. Push the complete project to GitHub
9. Deploy the site online using Vercel
